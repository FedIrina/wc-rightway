================================================================================
СПРАВКА: КОНТРАКТЫ API (не менять без причины)
================================================================================

Сырой ответ (rightwayAjaxPost), success проверяет сценарий:
  - sendCode, sendContactCode
  - getCardSummary, getCustomerContacts, getCustomersContacts

reject при success:false (rwPostOrReject):
  - getRwCustomerCards, editCommunicationData, editContactData
  - createCustomer, createContact, editCustomerData

Особый случай:
  - getRwToken → resolve строка-токен, reject текст, catch сеть

Ветвление при success:true (только сценарий):
  - getCustomersContacts: data.length 0 / 1 / >1
